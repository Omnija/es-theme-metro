# Metro Theme

For use with EmulationStation (http://www.emulationstation.org/)

### Notes

### Featured Sizes
* 1080p
* 720p

## Install Themes
Transfer es_theme_metro to /etc/emulationstation/themes/

or

/home/pi/.emulationstation/themes/

Open Menu -> Select UI Settings ->  Theme Set
